Keyboard Shortcuts

  • Toggle Mouse Pass-through:
    Press Ctrl+Alt+` to allow mouse clicks to "pass through" this window to the applications behind it. Press the key combination again to restore normal mouse interaction.
  • Minimize Window Content:
    Press Ctrl+Alt+Z to minimize the content section of the application. Press the key combination again to restore content section size.
  • Resize Window:
    Press Ctrl+Shift+Arrow Keys to resize the application window.
  • Move Window:
    Press Ctrl+Alt+Arrow Keys to move the application window.
  • Clear DPS:
    Press Ctrl+Alt+C to clear all DPS data.
